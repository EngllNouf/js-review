Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using various elements and tags.

Note: HTML is not a programming language; it's a markup language that tells browsers how to structure web pages.
Basic HTML Example 

<!DOCTYPE html>
<html>
    <head>
        <title>My First HTML Page</title>
    </head>
    <body>
        <h1>Welcome to HTML!</h1>
        <p>This is my first web page.</p>
    </body>
</html>

Welcome to HTML!

This is my first web page.

Try It Yourself!

The best way to learn HTML is by practicing. Try creating your first HTML page using the structure above.

HTML Basic Structure

Every HTML document must start with a document type declaration: <!DOCTYPE html>.

The HTML document itself begins with <html> and ends with </html>.

HTML Structure Explained 

<!DOCTYPE html>                   <!-- Document type declaration -->
<html>                          <!-- Root element -->
    <head>                      <!-- Contains metadata -->
        <title>Page Title</title>
        <meta charset="UTF-8">
    </head>
    <body>                      <!-- Contains visible content -->
        <!-- Your content here -->
    </body>
</html>
Important: The <head> section contains metadata about the document, while the <body> section contains the visible content.
